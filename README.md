# Beitrag
東洋大学情報連携学部情報連携学科(INAD)の生徒限定のコミュニケーションサイト。学部生の持つINIADのメールアドレスでユーザー登録することで、利用できるスマートフォン用のサイトです。  制作期間：約1ヶ月（2021年6月から2021年7月）
